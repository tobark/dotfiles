# Hyunjoon's dotfiles

## Installation
```sh
git clone https://github.com/tobark/dotfiles.git
```

### Zsh
```sh
ln -s ~/dotfiles/.zshrc ~/.zshrc
```

### Neovim
```sh
ln -s ~/dotfiles/.vimrc ~/.config/nvim/init.vim
```
